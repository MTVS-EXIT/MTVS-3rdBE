## 프로젝트 소개 
"재난 대피 시뮬레이션" 프로그램 <p>
이 프로젝트는 메타버스 아카데미의 XR-언리얼, 백엔드(back-end), AI, TA 팀이 협업하여 진행한 프로젝트입니다. <br>

### 기획 내용 
제한적인 시야에서 드론의 안내를 받아 화재로 뒤덮인 건물을 탈출하는 내용으로 진행됩니다.<p><br>
